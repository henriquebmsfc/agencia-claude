# /atualizar — Sincronizar Realidade

Sincroniza os arquivos de memória com mudanças no negócio — produtos novos, preços, foco, e qualquer coisa relevante que tenha mudado.

## Instruções

1. **Ler arquivos de memória:**
   - Leia `_memoria/empresa.md`
   - Leia `_memoria/preferencias.md`
   - Leia `_memoria/estrategia.md`

2. **Coletar mudanças:**
   - Pergunte: "O que mudou no negócio desde a última atualização? Pode ser novo produto, preço novo, mudança de foco, ou qualquer coisa relevante."

3. **Identificar seções afetadas:**
   - Com base na resposta, identifique quais seções de quais arquivos precisam ser atualizadas
   - Liste as mudanças antes de aplicar

4. **Validar com o dono:**
   - Mostre: "Vou atualizar [arquivo] — seção [seção]. Fica assim: [novo conteúdo]. Pode confirmar?"
   - Aguarde confirmação

5. **Atualizar arquivos:**
   - Aplique as mudanças nos arquivos confirmados

6. **Atualizar data:**
   - Atualize também a data de última atualização em `_memoria/estrategia.md`

7. **Sugerir versionamento:**
   - "Mudanças aplicadas! Use `/salvar` para versionar as mudanças no GitHub."

## Campos que podem mudar

- **Empresa:** novos produtos, mudança de preços, novos clientes ideais
- **Preferências:** tom de voz, exemplos de comunicação
- **Estratégia:** foco do momento, meta do mês
